Instalasi debian Linux CLI (comand line interface)

langsung ajalah gausah ribet capek ngetik nya

1. tahap pertama Pada Menu Instaler Pilih Bagian “INSTALL”

2. kemudian ditahap yang kedua yaitu Select languange, pilih bagian default saja yaitu “English”

3. lanjut ditahap yang ketiga yaitu Select country, ini menyesuaikan tempat anda. Disini saya lokasinya diindonesia

kalo bahasa indonesia nya gk ada pilih other dulu trus masuk ke continent (benua) asia, entar juga ada

4. pada tahap yang keempat yaitu Local configure, pilih di US

5. kemudian pada bagian Keymap, kita setting default saja yaitu America English

6. Kemudian pada tahap keenam bagian kofigurasi network kita “no” saja, kemudian klik “ do not configure the network”

7. ditahap yang ketujuh ini kalian diminta untuk mengisi Hostname dan Password, kalian isi sesuka kalian


8. ditahap yang kedelapan ini kalian diminta untuk mengisi New user dan password user, kalian isi sesuka kalian


9. Kamudian pada bagian select city, disini saya memilih bagian "western”

untuk city kalo gk sama kaya di gambar, aman ajah, soalnya tergantung kalian pilih negara/country apa pas di awal

10. Lalu pada pilihan Partitioning method nya kita pilih bagian “use entire disk”

11. Masih lanjut, partitionig scheme kita pilih bagian “separate /home, /var, and /tmp partitions. Jika sudah lalu klik “finish” saja

12. Pada write the change to disk kita “yes” saja


13. Scan another cd & network miror bisa di skip saja dengan tekan “NO”


14. Participate in the package usage survey kita pilih “YES”


15. Tahap ini kalian pilih software yang kalian mau langsung di install, cara menambahkan/menghilangkan bintang dengan cara klik “sapce”

16. Padah tahap group boot loader kita pilih “YES” saja kemudian pilih yang /dev/sda

17. Installasi telah berhasil langsung klik “continue” untuk melanjutkan

Udah dech, btw untuk nampilin gambar debian di terminal, install neofetch dulu ya, comand nya juga neofetch
