Domain Name Server Tutorial

Katanya SUSAH Emang iya?

Bayangkan DNS server itu seperti pelayan di restoran. Anda datang dan bilang, “Saya mau Nasi Goreng,” yang mana itu seperti nama domain. Pelayan (DNS server) tahu persis di mana ‘Nasi Goreng’ itu berada di dapur (alamat IP) dan membawa pesanan Anda ke meja. Jadi, DNS server ini membantu komputer Anda menemukan tempat yang tepat di internet dengan menggunakan nama yang mudah diingat, bukan deretan angka yang sulit dihafal. Berikut adalah keuntungan menggunakan Dns Server:

  • Mempercepat Akses Web.DNS server meng-cache hasil pencarian, memperpendek waktu akses untuk kunjungan ulang ke website yang sama.
  • Memfasilitasi Pengiriman Email. Dengan fitur MX record, DNS memastikan email sampai ke tujuan yang benar, ini akan berhubungan dengan Mail Server Nantinya
  • Menerjemahkan alamat ip. DNS mengubah nama domain yang mudah diingat (seperti ‘google.com’) menjadi alamat IP numerik yang dibutuhkan untuk koneksi komputer.
